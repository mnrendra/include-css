# @mnrendra/include-css
To copy all of css files into the production (ex: lib) directory

## Features
* includeCSS

## Installation
```c
npm i @mnrendra/include-css
```

## Example
```c
const includeCSS = require('@mnrendra/include-css')
const allFiles = includeCSS('~/')
console.log(allFiles)
```
